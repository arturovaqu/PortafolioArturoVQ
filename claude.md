# Proyecto: Portfolio Clone - ARTURO VALENCIA
Guía de estilo y reglas de desarrollo para el agente.

## Tech Stack
- Framework: React (Vite)
- Estilos: Tailwind CSS
- Animaciones: Framer Motion
- Iconos: Lucide React
- Assets: Localizados en `design_docs/Images_web/`

## Design Tokens (Extraídos de Figma)
- **Fondo:** `#000000` (Negro puro)
- **Acento:** `#D4F27A` (Verde lima brillante)
- **Texto Primario:** `#FFFFFF`
- **Texto Secundario:** `#94a3b8` (Slate-400)
- **Cards/Inputs:** Fondo `#111111` con bordes sutiles.
- **Tipografía:** Inter o Geist (Sans-serif moderna).

## Reglas de Oro con Superpowers
1. **Analizar antes de actuar:** Usa siempre `superpowers:brainstorming` para comparar `HomePage.png` y `HomePageMovil.png`.
2. **Planificación:** Cada tarea compleja requiere un `superpowers:writing-plans`.
3. **Fidelidad Visual:** El layout debe ser idéntico al de las capturas. No uses textos de relleno ("Lorem Ipsum"), extrae el texto real de las imágenes de `design_docs`.
4. **Mobile First:** El diseño debe ser perfecto en móvil antes de escalar a escritorio.
